三数之和、四数之和

发表于 2021-03-31 分类于 算法题 阅读次数： Valine：

三数之和: 给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

示例 1：

输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]

示例 2：

输入：nums = []
输出：[]

示例 3：

输入：nums = [0]
输出：[]

方法一(双指针)

一二指针不动，第三指针移动

先对当前数组从小到大排序,这里定义三个指针

让第一个指针从头循环遍历该数组，每一次遍历先让第二指针指向第一指针的下一个位置，再让第二个指针循环遍历数组

第二指针遍历时让第三指针从最后开始走，如果三个指针的值相加大于0，就让第三指针左移，直到等于0，

将结果添加到集合，并最后返回该集合

注意: 指针要去重

一指针不动，二三指针同时移动

二三指针这里变成左右指针，左指针一定要小于右指针

如果三个数相加小于0，左指针移动，大于0，右指针移动，等于0就将三个数添加到list集合，并同时移动左右指针

最后将集合返回

四数之和

给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 a，b，c 和 d ，使得 a + b + c + d 的值与 target 相等？找出所有满足条件且不重复的四元组。

注意：答案中不可以包含重复的四元组。

示例 1：

输入：nums = [1,0,-1,0,-2,2], target = 0
输出：[[-2,-1,1,2],[-2,0,0,2],[-1,0,0,1]]

示例 2：

输入：nums = [], target = 0
输出：[]

思路分析

方法与三数之和相同，再三数之和的基础上再包一个for循环，看作是第四个数

定义四个指针 a ，b ，c ，d，

a在最外层，从头遍历数组，b为下一层，从a的下一个数开始遍历数组，c与d为第三层，看作左右指针同时移动。。。。。。(参考两数之和改进后的方法)
